# Tech Stack

- Lua 5.1 (Mudlet embedded interpreter). No rocks, no build system, no package manager.
- Target runtime: Mudlet client; scripts are loaded into a Mudlet profile manually.
- No test framework in repo (one ad-hoc `sentinel_test.lua`).
- Docs in Markdown alongside each module.
- Dev OS: Windows; shell: PowerShell 7.
